Khái niệm

  • Bộ chọn (Selectors): dùng để chọn phần tử HTML cần áp dụng CSS.
  • Cú pháp (Syntax): gồm bộ chọn, cặp ngoặc nhọn và các thuộc tính–giá trị.
  • Ưu tiên (Specificity): xác định thứ tự áp dụng khi nhiều quy tắc cùng ảnh hưởng.
  • Kế thừa (Inheritance): một số thuộc tính CSS tự động áp dụng cho phần tử con.

Bước thực hành nhanh

  1. Tạo file style.css và viết mã CSS.
  2. Liên kết file CSS vào HTML bằng thẻ <link>.
  3. Mở trang trong trình duyệt để kiểm tra kết quả.
